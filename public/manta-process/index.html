repo.box · manta pipeline

From raw dive footage to loopable manta silhouettes

We reused the same extraction system we built for jellyfish: clip selection, contrast shaping, brightness-frame extraction, then a ping-pong loop pass for seamless playback in hero animations.

Step 1

Select a clean pass

Trimmed a 6s segment where one manta crosses with clear body definition.

ffmpeg -ss 4 -t 6 -i source.mp4 -vf "crop=720:860:0:0,fps=12" 01-clip-raw.mp4

Step 2

Preprocess for silhouette extraction

Removed color, boosted contrast, and smoothed water noise to isolate manta shape signal.

ffmpeg -i 01-clip-raw.mp4 -vf "eq=contrast=2.4:brightness=-0.08:saturation=0,gblur=sigma=1.2,curves=all='0/0 0.28/0.03 0.55/0.72 1/1'" 02-clip-preprocessed.mp4

Step 3

Build a threshold mask

Mapped darker values to foreground mask to pull manta contours from bright water background.

Step 4

Extract brightness frames to JSON

Converted processed video into a frame grid used directly by canvas runtime.

node tools/gif-to-ascii.js 02-clip-preprocessed.mp4 public/manta.json --cols=120 --rows=56

Then made it loop-safe with ping-pong sequencing to remove hard cuts.

manta-loop.json = frames + reverse(frames[1..-2])

Final

Loop preview

Forward + reverse pass creates a seamless, majestic cycle suitable for hero-layer reuse.
